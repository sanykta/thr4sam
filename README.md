Overview

This project performs Exploratory Data Analysis (EDA) on a Netflix dataset to uncover trends in content distribution, genres, countries, and release patterns.

The goal is to understand how Netflix’s content strategy has evolved using data-driven insights.

Objectives
Analyze the distribution of Movies vs TV Shows
Identify trends in content production over the years
Explore the most common genres and categories
Understand country-wise contribution to Netflix content
Derive insights from real-world entertainment data

Dataset
Source: Kaggle (Netflix Titles Dataset)
Contains information about:
Title
Type (Movie/TV Show)
Country
Genre
Release Year
Duration
Director


Tools & Technologies
Python
Pandas (data cleaning & analysis)
Matplotlib / Seaborn (visualization)


Data Cleaning Steps
Handled missing values (e.g., country, director)
Removed or labeled unknown entries
Standardized categorical values
Processed multi-value fields (like country, genres)
Checked for duplicates and inconsistencies


Key Questions Explored
What is the ratio of Movies vs. TV Shows?
How has content production changed over time?
Which countries contribute the most content?
What are the most common genres?
Are there noticeable trends in recent years?


Key Insights
Movies do dominate the Netflix catalog compared to TV Shows.
Content production increased significantly after 2010 but suffered a drastic drop during the pandemic.
United States and India are among the top contributors


Analysis Approach
Data loading and inspection
Data cleaning and preprocessing
Exploratory analysis using statistical summaries
Visualization of trends and distributions
Insight generation and interpretation


Conclusion
This project demonstrates how data analysis can uncover meaningful patterns in entertainment content, helping understand platform strategies and challenges.
